How to install?

Requirements:

  • Stable Internet Connection
  • Mozilla Firefox 62.x

Understanding the Code Structure:

  • Webpages - HTML code for all the webpages are in the root folder which is VisitPedia
  • CSS - CSS for all the webpages are in root/css and assets/css
  • JavaScript - JS for the visualizations and animations are within js folder in root/js and root/assets/js
  • Images - All the images used for this project are in root/images
